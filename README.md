# Interactive Voting Poll App

A simple and dynamic voting poll application that allows users to vote for their preferred option and instantly see updated results.

This project demonstrates JavaScript event handling, DOM manipulation, state management, and real-time UI updates without using a backend.

## Features

- Multiple voting options
- Real-time vote count updates
- Percentage calculation for each option
- Dynamic result display
- Prevention of duplicate voting (if implemented)
- Responsive modern design

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)

## what This Project Demonstrates

- DOM manipulation
- Event-driven programming
- Data storage using arrays/objects
- Dynamic rendering of UI elements
- Basic algorithm for percentage calculation
- State management logic

## Project Structure

```
/project-folder
│── index.html
│── style.css
│── script.js
```

## How It Works

1. User selects a voting option.
2. JavaScript updates the vote count.
3. Percentages are calculated dynamically.
4. Results are rendered instantly without refreshing the page.
